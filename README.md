# Sales_Dashboard

This repository contains an interactive **{Tableau Dashboard}** that analyzes year-over-year sales performance for **2022**, focusing on profitability, product subcategories, customer behavior, and weekly sales trends.  
The dashboard delivers actionable insights for decision-makers through KPI tracking, subcategory comparison, and trend highlighting.

---

## Overview

The **Sales Performance Dashboard** provides a rich visualization of how sales, profit, and product categories performed throughout **2022**.  
It compares results with the previous year and highlights periods of exceptional performance â€” helping analysts and business leaders evaluate growth, seasonality, and profitability.
